# Scientific Calculator Project

## Project Overview

This project is a scientific calculator application designed to perform both basic and advanced mathematical operations. The calculator provides an easy-to-use interface for executing arithmetic calculations as well as scientific functions.

The project demonstrates fundamental programming concepts, logical problem solving, and mathematical computation.

---

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Scientific functions such as powers, square roots, and trigonometric calculations
- Input validation and error handling
- Clear and structured code implementation
